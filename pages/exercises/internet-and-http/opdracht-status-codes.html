Opdracht HTTP Status Codes

Op deze webpagina staan een aantal links, voer volgende opdracht uit voor elke link:

  • Open de Netwerk-tab van je browser Developer Tools en klik op de link.
  • Klik in de Netwerk-tab op het juiste request en bekijk de Headers.
  • Neem een screenshot van het General venster en voeg deze toe aan de source code van dit project onder de juiste link.
  • Voeg een korte beschrijving toe van de status code en wat deze betekent.
