Elementen & Attributen: Opdracht 1: Boodschappenlijst

In de map pages > exercises > javascript > elements > task-1 vind je de bestanden die je nodig hebt voor deze opdracht.

Opgelet: voor deze opdracht mag je NIETS wijzigen in de HTML code. Je zal enkel de JavaScript code moeten aanpassen.

Requirements:

  • Een klik op een ingrediënt maakt het rood - geen class toevoegen!
  • Een klik op de 'Voeg toe' knop, voegt het ingrediënt toe aan de lijst.
  • Een klik op de 'Wis alles' knop, verwijdert alle ingredienten.

Je kan je opdracht lokaal doen draaien door in de map task-1 met de rechtermuisklik op index.html te kiezen voor 'Open with Live Server'.

Beantwoord na je implementatie volgende vragen (schrijf je antwoorden in de source code van deze pagina):

  • Nadat je een ingrediënt hebt toegevoegd, kun je dan een oorspronkelijk ingrediënt nog rood maken? Waarom wel/niet?
    ...
  • Kun je een nieuw ingrediënt ook rood maken door er op te klikken? Waarom wel/niet?
    ...
