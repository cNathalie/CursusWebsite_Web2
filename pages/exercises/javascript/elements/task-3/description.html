Elementen & Attributen: Opdracht 3: Dynamische afbeelding

In de map pages > exercises > javascript > elements > task-3 vind je de bestanden die je nodig hebt voor deze opdracht.

Schrijf een webpagina met een inputveld, een knop en een afbeelding.

Requirements:

  • Initieel toon je een hardgecodeerde afbeelding die je zelf kiest.
  • In het inputveld kan de gebruiker een url ingeven van een nieuwe afbeelding.
  • Een klik op de 'Wijzig' knop, zorgt ervoor dat de afbeelding met die van de url wordt vervangen.

Je kan je opdracht lokaal doen draaien door in de map task-3 met de rechtermuisklik op index.html te kiezen voor 'Open with Live Server'.
