Forms: Opdracht Gemeenten

Requirements:
  • Schrijf een programma dat de gebruiker herhaaldelijk om een gemeente vraagt.
  • Op deze manier verzamelt het programma een lijst van gemeenten.
  • Wanneer de gebruiker op 'cancel' duwt of het woord 'stop' ingeeft, sorteert het programma de lijst van ingegeven gemeenten alfabetisch.
  • De gesorteerde gemeenten worden vervolgens een voor een gebruikt in een option element waarmee een select element wordt opgevuld.
Technical notes:
  • Maak gebruik van de ingebouwde window.prompt() methode. Meer info.
  • Controleer de return waarde van de prompt en verwerk correct naargelang de waarde.
  • Voorzie in je HTML code een select element met het id 'selGemeenten'.
  • Voeg in je JS code per gemeente een option element toe.
