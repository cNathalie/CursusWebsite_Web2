JSON Opdracht: Studentengegevens

Maak een JSON-bestand met jouw eigen studentengegevens en gebruik dit object om op de homepage van deze cursus-website je gegevens in te laden.

Je zal hiervoor een JavaScript-bestand moeten aanmaken en dit bestand linken aan de homepage van deze cursus-website. Je JavaScript bestand zal het JSON-object moeten binnenhalen met de fetch functie en daarna de gegevens inladen in de HTML van de homepage.
