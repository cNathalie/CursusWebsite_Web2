Objecten opdracht: Bibliotheek

Met je opgedane kennis over JavaScript objecten kan je nu perfect een eigen bibliotheek starten.

Requirements JS:

  • Zorg ervoor dat je 'Book' objecten kan aanmaken met volgende eigenschappen:
    • title
    • author
    • genre
    • year
  • Maak een 'Library' object aan met volgende eigenschappen:
    • name (een creatieve naam voor je bib)
    • books (array)
    • addBook (methode!)
    • displayBooks (methode!)
  • De methode addBook moet een Book object toevoegen aan de books array.
  • De methode displayBooks moet alle boeken in de library tonen in de console.

Requirements HTML:

  • Maak een HTML pagina met input velden voor alle Book eigenschappen en een knop 'Voeg toe'.
  • Wanneer de gebruiker op de knop klikt, moet er een nieuw Book object aangemaakt worden met de waarden die de gebruiker heeft ingegeven in het input veld.
  • Voeg dit Book object toe aan de library.
  • Voeg een tweede knop toe die de methode displayBooks aanroept (het loggen van de Book eigenschappen van elk Book in Console volstaat).

Technical notes:

  • Maak gebruik van een constructor functie om Book objecten aan te maken.
  • Zorg ervoor dat er al drie boeken hard coded in je bibliotheek zitten.
  • Denk aan de stack functies voor het implementeren van addBook.
  • Bij displayBook maak je gebruik van een loop om over al je boeken heen te gaan.
  • Toon alle boeken in de console met een mooie opmaak aan de hand van de template literals.

Veel succes!
