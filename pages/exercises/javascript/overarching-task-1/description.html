Overkoepelende Opdracht 1: Color Picker

In deze opdracht ga ja zelf een ColorPicker maken. Dit is een tool die de gebruiker ervan toelaat om zelf een kleur te mengen. De gebruiker kan de kleur kiezen door de RGB waarden in te geven.

Je hebt hiervoor al je tot hiertoe opgedane kennis van Programmeren Basis, HTML, CSS en JavaScript nodig.

Requirements:

  • De gebruiker kan de RGB waarden ingeven aan de hand van drie sliders. Deze siders zijn beperkt tot een waarde tussen 0 en 255.
  • Naast elke slider ziet de gebruiker de numerieke waarde waarop de slider op dat moment is ingesteld.
  • De gebruiker kan de kleur die hij heeft samengesteld zien in een vierkantje.
  • Telkens wanneer de gebruiker een slider verandert, wordt de kleur opniew gemengd en weergegeven.

Technical notes:

  • Maak gebruik van één update-functie die:
    • De waardes van de sliders uitleest.
    • De ingegeven waardes toont naast de sliders.
    • De kleur (style) van het vierkantje aanpast.

Je voegt zelf alle nodige files (.html, .css, .js) toe onder de map pages > exercises > javascript > overarching-task-1.
