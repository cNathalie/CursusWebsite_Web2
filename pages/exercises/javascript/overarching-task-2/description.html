Overkoepelende Opdracht - 2: Uitbreiding Color Picker

Maak in deze folder een kopie van de bestanden van Overkoepelende Opdracht - 1.

In deze opdracht ga je de Color Picker uitbreiden met de volgende functionaliteiten.

Requirements:
  • Voeg rechts van de colorpicker component een 'Save' knop toe.
  • Wanneer de gebruiker op de 'Save' knop klikt, wordt er een kopie van de gekleurde kader ('swatch') toegevoegd onder de component.
  • In de swatch wordt de kleur weergegeven die de gebruiker heeft geselecteerd.
  • In de rechterbovenhoek van elke swatch staat een 'Delete' knop.
  • Wanneer de gebruiker op de 'Delete' knop klikt, wordt de swatch verwijderd.
  • Wanneer de gebruiker op een swatch klikt, wordt de kleur van de kader terug ingesteld in de colorpicker.
Technical notes:
  • Wanneer de gebruiker op de 'Save' knop klikt, wordt er een kopie van de gekleurde kader toegevoegd onder de component
    • Voorzie de delete knop maar koppel nog geen click gedrag aan de swatchof de delete knop
    • De layoutis geen prioriteit maar als je je geroepen voelt: de delete knop wordt absoluut gepositioneerd en je krijgt hem in de rechterbovenhoek door de top en right CSS-properties in te stellen
  • Wanneer de gebruiker op een gekleurde kader klikt, wordt de kleur van de kader terug ingesteld in de colorpicker.
    • De waarde van de sliders voor een bewaarde kleur, stop je in drie custom attributen (data-red, data-green en data-blue) van het swatch element.
    • In de click event listener achterhaal je met event.target het swatch element, vraagt de waarden van diens 3 custom attributen op, gebruikt ze om de sliders te herstellen en roept tenslotte update() op
  • Wanneer de gebruiker op de 'Delete' knop klikt, wordt de gekleurde kader verwijderd.
    • Hier ligt een moeilijkheid omdat de delete knop een child is van het swatch element. Een klik op de delete knop zal dus wegens event bubbling ook de click event listener van het swatch element activeren
    • Je zal dus in die event listener een onderscheid moeten maken tussen een directe klik op de swatch en een klik op de delete knop...
