DOM: Stamboomstructuur

Introductie Stamboom-structuur Elementen opvragen Formulieren Opdracht

Ja kan de DOM-Tree een beetje vergelijken met een familieboom.

Top-node

De top-node in de DOM-tree is altijd de document node. Deze stelt het volledige document voor. Je tekent het in een schema net boven het html element.

Root

Het top-level element van een document waarvan alle andere elementen descendants (kinderen) zijn. In dit geval is dat het html element.

Parent

Een parent is een element dat een ander element bevat. In een familieboom is dat bijvoorbeeld een ouder die kinderen heeft.

Child

Een child is een element dat door een ander element bevat wordt. In een familieboom is dat bijvoorbeeld een kind dat ouders heeft.

Sibling

Een sibling is een element dat dezelfde parent heeft als een ander element. In een familieboom zijn dat bijvoorbeeld broers en zussen.

Descendant

Een descendant is een element dat een parent heeft. In een familieboom zijn dat alle directe kinderen en kleinkinderen.

Ancestor

Een ancestor is een element dat een child heeft. In een familieboom zijn dat bijvoorbeeld grootouders.
