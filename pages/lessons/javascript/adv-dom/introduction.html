Wat is de DOM?

De DOM, of Document Object Model, is een object georiÃ«nteerde representatie van een HTML document. Het is een boomstructuur van nodes die elk een deel van het document voorstellen. De DOM is een manier om een HTML document te representeren zodat we het kunnen manipuleren met JavaScript.

De DOM bestaat uit verschillende soorten nodes. Een node kan het volgende zijn:

  • Document: de top node in de DOM-tree. Het heeft het volledige document in zich.
  • Element nodes: Elke HTML-tag is een element in de boom.
  • Text nodes: De tekstinhoud van een bepaald element is altijd een text-node. Witruimte wordt ook voorgesteld als een text-node!
  • Attribute nodes: Een attribuut van een node is bereikbaar via he element, maar wordt niet in de de boomstructuur getekend.
