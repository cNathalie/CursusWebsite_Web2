Nieuwe nodes moeten toegevoegd worden aan de DOM om zichtbaar te worden.

appendChild()

Met de appendChild() methode kunnen we een node toevoegen aan een andere node.

De methode neemt één argument: de node die je wil toevoegen.

De node die je wil toevoegen wordt als laatste kind toegevoegd aan de node waarop je de appendChild() methode uitvoert.

Voorbeeld:

We maken een nieuwe button aan en voegen die toe net onder deze zin.


// We maken een nieuwe knop aan
const newBtn = document.createElement('button');

// We halen de lege paragraaf net boven dit blok op
const emptyPara = document.querySelector('#para');

// We voegen de knop toe aan de paragraaf
emptyPara.appendChild(newBtn);

Je ziet (amper) een nieuwe knop verschijnen, maar hij heeft geen tekst in zich.

We kunnen de tekst van de knop aanpassen door een tekstnode toe te voegen aan de knop. 


// We maken een nieuwe tekstnode aan
const btnText = document.createTextNode('Klik hier');

// We voegen de tekstnode toe aan de knop
newBtn.appendChild(btnText);

removeChild()

Met de removeChild() methode kunnen we een node verwijderen uit de DOM.

De methode neemt één argument: de node die je wil verwijderen.

Voorbeeld:

We verwijderen de knop die we net hebben toegevoegd.


// We halen de knop op
const btn = document.querySelector('button');

// We halen de parent node van de knop op
const parent = btn.parentNode;

// We verwijderen de knop uit de parent node
parent.removeChild(btn);

Opgelet!

De methodes zijn enkel beschikbaar op de parent node van de node die je wil toevoegen of verwijderen.

parent.METHODE(child)

Als je in een div een element wil toevoegen, moet je dus eerst de div ophalen, dan je element aanmaken en dan de appendChild() methode uitvoeren op de div.
