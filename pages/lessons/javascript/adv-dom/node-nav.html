Je kan navigeren tussen nodes in de DOM. Dit kan je doen met de volgende properties:

parentNode

De parent node van de huidige node.


// We halen deze div op
const node = document.querySelector('#my-div');
// We halen de parent node op
const parentNode = node.parentNode;

console.log(parentNode);

childNodes

De children nodes van de huidige node.


// We halen deze div op
const node = document.querySelector('#my-div');
// We halen de children nodes op
const childNodes = node.childNodes;

console.log(childNodes);

firstChild

Het eerste child node van de huidige node.


// We halen deze div op
const node = document.querySelector('#my-div');
// We halen het eerste child node op
const firstChild = node.firstChild;

console.log(firstChild);

lastChild

Het laatste child node van de huidige node.


// We halen deze div op
const node = document.querySelector('#my-div');
// We halen het laatste child node op
const lastChild = node.lastChild;

console.log(lastChild);

nextSibling

Het volgende sibling node van de huidige node.


// We halen deze div op
const node = document.querySelector('#my-div');
// We halen het volgende sibling node op
const nextSibling = node.nextSibling;

console.log(nextSibling);

previousSibling

Het vorige sibling node van de huidige node.


// We halen deze div op
const node = document.querySelector('#my-div');
// We halen het vorige sibling node op
const previousSibling = node.previousSibling;

console.log(previousSibling);

Opgelet

Alle bovenstaande properties geven nodes van alle types terug. We zijn echter meestel enkel geinteresseerd in element nodes (HTML-Elementen). Daarvoor heb je onderstaande properties ter beschikking:

children

De children element nodes van de huidige node.


// We halen deze div op

// We halen de children element nodes op
const children = node.children;

console.log(children);
