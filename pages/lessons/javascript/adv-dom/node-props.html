Een Node heeft veel properties. Je kan deze bekijken via de Developer Tools in je browser, in het tabblad "Elements" en subtabblad "Properties". We bespreken de meest gebruikte:

nodeName

De naam van de node, altijd in hoofdletters.


// We halen deze div op
const node = document.querySelector('#my-div');
console.log(node.nodeName);

nodeValue

De waarde van de node.

  • Element nodes: null
  • Attribute nodes: de waarde van de attribute
  • Text nodes: de tekst van de node

// We halen deze div op
const node = document.querySelector('#my-div');
console.log(node.nodeValue);

nodeType

Het type van de node.


// We halen deze div op
const node = document.querySelector('#my-div');
console.log(node.nodeType);
