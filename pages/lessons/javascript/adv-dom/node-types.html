Nodes bestaan in verschillende types.

Types

Je kan het type van een node opvragen via de nodeType property. Deze methode geeft een nummer terug dat het type van de node aangeeft. Voorbeeld: 


// We halen deze div op
const node = document.querySelector('#my-div');

// We loggen het type van de node
console.log(node.nodeType);

Dit zal 1 loggen naar de console, wat het type is van een element node.

Hier zijn de codes van nodes die wij leerden kennen:

  • Element node: 1
  • Attribute node: 2
  • Text node: 3
  • Document node: 9
