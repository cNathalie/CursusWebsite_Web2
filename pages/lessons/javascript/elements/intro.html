We zagen eerder al dat we met JavaScript elementen kunnen selecteren aan de hand van een aantal GET methodes op het document object.

Elementen selecteren

We kennen reeds volgende methodes:

  • getElementById('id')
  • getElementsByTagName('tag')
  • getElementsByClassName('class')

Deze methodes zijn echter niet de enige manier om elementen te selecteren. We kunnen ook elementen selecteren met behulp van:

  • querySelector('selector')
  • querySelectorAll('selector')

De 'selector' is een CSS selector, we komen hier later op terug.
