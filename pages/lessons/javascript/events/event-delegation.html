JavaScript Events: Delegation

Intro Events Event Listeners Event Object Methodes & Bubbling Event Delegation Oefeningen

Event delegation is een techniek die je kan gebruiken om events te behandelen op meerdere elementen met slechts één event handler. Dit is handig wanneer je meerdere elementen hebt die dezelfde event listener nodig hebben.

In plaats van een event listener toe te voegen aan elk element, kan je een event listener toevoegen aan een parent element en dan de events delegeren naar de children elementen.

Dit is handig wanneer je bijvoorbeeld een lijst hebt met items die allemaal dezelfde actie moeten uitvoeren wanneer erop geklikt wordt. In plaats van een event listener toe te voegen aan elk item, kan je een event listener toevoegen aan de parent lijst en dan de events delegeren naar de children items.
