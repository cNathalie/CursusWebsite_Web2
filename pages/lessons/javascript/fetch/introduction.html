Fetch: Introductie

Introductie Fetch GET

Fetch is een API die het mogelijk maakt om HTTP requests te maken in JavaScript.

We leren in dit stukje kort Fetch kennen, maar we gaan er niet diep op in.
Uitdieping van Fetch komt in een latere cursus aan bod.

Fetch API

Fetch is een promise-based API. Dit betekent dat het werkt met promises. Een promise is een object dat een waarde vertegenwoordigt die mogelijk beschikbaar is in de toekomst.

Letterlijk een "belofte" dat er een waarde beschikbaar zal zijn. Maar zoals elke belofte kan deze worden gehouden of verbroken.

Fetch is standaard beschikbaar in de meeste moderne browsers. Het is echter niet beschikbaar in Internet Explorer.
