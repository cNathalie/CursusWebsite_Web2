Er zijn verschillende input varianten die je kan gebruiken in een invulformulier.

Text Input

Een tekstveld is een van de meest voorkomende input types. Hier kan de gebruiker tekst invoeren. We halen de waarde van een tekstveld op met de .value property. 


HTML
<input type="text" placeholder="Voer hier tekst in" />

JavaScript
const input = document.querySelector('input');
const inputValue = input.value;

Password Input

Een wachtwoordveld is een tekstveld waarvan de ingevoerde tekst verborgen wordt. We halen de waarde van een wachtwoordveld op met de .value property. 


HTML
<input type="password" placeholder="Voer hier je wachtwoord in" />

JavaScript
const password = document.querySelector('input[type="password"]');
const passwordValue = password.value;

Checkbox

Een checkbox is een vakje waar de gebruiker een vinkje in kan zetten. We halen de waarde van een checkbox op met de .checked property. 


HTML
<input type="checkbox" id="chk"/>




JavaScript
const checkbox = document.querySelector('#chk');
const isChecked = checkbox.checked;
console.log(isChecked);

Radio Button

Een radio button is een knop waar de gebruiker uit meerdere opties kan kiezen. We halen de waarde van een radio button op met de .checked property. 


HTML
<input type="radio" name="colors" value="red" />





    
    




    
    




JavaScript
const checkedRadioButton = document.querySelector('input[name="colors"]:checked');
console.log(checkedRadioButton.value);

Select

Een select element is een dropdown menu waar de gebruiker een optie uit kan kiezen. We halen de waarde van een select element op met de .value property. 


HTML: Single select
<select name="fruits" id="fruits">
    <option value="apple">Appel</option>
    <option value="banana">Banaan</option>
</select>



JavaScript: Single select
const select = document.querySelector('#fruits');
const selectedOption = select.value;
console.log(selectedOption);

HTML: Multiple select
<select name="vegetables" id="vegetables" multiple>
    <option value="carrot">Wortel</option>
    <option value="cucumber">Komkommer</option>
    <option value="tomato">Tomaat</option>
</select>


(Je moet CTRL inhouden om er meerdere te selecteren.)

JavaScript: Multiple select
const selectedOptions = document.querySelectorAll('#vegetables option:checked');
console.log(selectedOptions);
