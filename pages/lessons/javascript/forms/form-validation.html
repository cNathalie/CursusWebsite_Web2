Wanneer we input krijgen van de gebruiker, dan willen we die valideren. Dit betekent dat we controleren of de input correct is. Als de input niet correct is, dan willen we de gebruiker laten weten wat er mis is. Dit kan bijvoorbeeld door een bericht te tonen of door de gebruiker te laten weten welke velden nog ingevuld moeten worden.

Strings valideren

Een van de meest voorkomende validaties is het controleren of een string niet leeg is. Dit kan je doen door de .length property van een string te gebruiken. Als de lengte van de string groter is dan 0, dan is de string niet leeg. 


const input = document.querySelector('input');
const value = input.value;

if (value.length > 0) {
    console.log('De string is niet leeg');
} else {
    console.log('De string is leeg');
}

Je kan ook een nullcheck doen:


const input = document.querySelector('input');
const value = input.value;

if (value) {
    console.log('De string is niet leeg');
} else {
    console.log('De string is leeg');
}

Strings vergelijken

Het kan ook zijn dat je strings met elkaar wil vergelijken. Dit kan op twee manieren:

  • Met de operatoren ===, !==, < en >.
  • Met de methode .localeCompare()

Hieronder zie je een voorbeeld van hoe je strings kan vergelijken met de operatoren: 


const input = document.querySelector('#comparing');
const value = input.value;

if (value === 'hello') {
    console.log('De string is gelijk aan "hello"');
} else {
    console.log('De string is niet gelijk aan "hello"');
}

Hieronder zie je een voorbeeld van hoe je strings kan vergelijken met de .localeCompare() methode: 


const input = document.querySelector('#comparing');
const value = input.value;

if (value.localeCompare('hello') === 0) {
    console.log('De string is gelijk aan "hello"');
} else {
    console.log('De string is niet gelijk aan "hello"');
}

De methode geeft een getal terug dat aangeeft welke tekst lexicografisch eerst komt:

  • Het restultaat is negatief indien string 1 voor string 2 komt.
  • Het resultaat is positief indien string 1 na string 2 komt.
  • Het resultaat is 0 indien de strings gelijk zijn.

Gelijkheid van twee waardes bepalen

Soms wil je controleren of twee waardes gelijk zijn aan elkaar. Dit kan je doen met de === operator. Deze operator controleert of de waardes gelijk zijn en of ze van hetzelfde type zijn. Als je enkel wil controleren of de waardes gelijk zijn, dan kan je de == operator gebruiken.

Hieronder staan twee inputvelden, het eerste is van het type text en het tweede is van het type number. Vul zowel in de de tekst-input als in de number-input dezelfde waarde in. Voer daarna de JavaScript-code uit in de Console-tab van de Developer Tools om te zien of de waardes gelijk zijn.

?= 

const input1 = document.querySelector('#input1');
const input2 = document.querySelector('#input2');

if (input1.value === input2.value) {
    console.log('De waardes zijn gelijk');
} else {
    console.log('De waardes zijn niet gelijk');
}

De waardes zijn gelijk als ze van hetzelfde type zijn en dezelfde waarde hebben. Inputvelden geven ALTIJD strings terug, ook wanneer het type number is!

Als we onze number-input waarde echt als een getal willen behandelen, moeten we het dus eerst omzetten naar een Number: 


const input1 = document.querySelector('#input1');
const input2 = document.querySelector('#input2');

// Zet de string om naar een Number
const numberValue = Number(input2.value);

if (input1.value === numberValue) {
    console.log('De waardes zijn gelijk');
} else {
    console.log('De waardes zijn niet gelijk');
}

We vergelijken nu dit gedrag, met dat van de == operator: 


const input1 = document.querySelector('#input1');
const input2 = document.querySelector('#input2');

if (input1.value == input2.value) {
    console.log('De waardes zijn gelijk');
} else {
    console.log('De waardes zijn niet gelijk');
}

// Zet de string om naar een Number
const numberValue = Number(input2.value);

if (input1.value == numberValue) {
    console.log('De waardes zijn gelijk');
} else {
    console.log('De waardes zijn niet gelijk');
}

We merken op dat de == operator de waardes als gelijk beschouwt, zelfs als ze van verschillende types zijn. Dit is niet altijd wenselijk, dus gebruik de === operator wanneer je zeker wil zijn dat twee waardes Ã©cht hetzelfde zijn.

Doorgaans gebruiken we de === operator, tenzij we een specifieke reden hebben om de == operator te gebruiken.

Gelijkheid van twee objecten bepalen

Als je twee objecten met elkaar wil vergelijken, dan kan je dit doen met de === operator. Deze operator controleert of de referenties naar de objecten gelijk zijn. Als je wil controleren of de waardes van de objecten gelijk zijn, dan moet je de afzonderlijke waardes van de objecten met elkaar vergelijken.

Hieronder zie je een voorbeeld van hoe je twee objecten kan vergelijken: 


const object1 = { name: 'Alice' };
const object2 = { name: 'Alice' };

if (object1 === object2) {
    console.log('De objecten zijn gelijk');
} else {
    console.log('De objecten zijn niet gelijk');
}

In dit geval zijn de objecten niet gelijk, omdat ze naar verschillende referenties in het geheugen wijzen. Als je de waardes van de objecten wil vergelijken, dan moet je de afzonderlijke waardes van de properties van de objecten met elkaar vergelijken: 


const object1 = { name: 'Alice' };
const object2 = { name: 'Alice' };

if (object1.name === object2.name) {
    console.log('De waardes van de objecten zijn gelijk');
} else {
    console.log('De waardes van de objecten zijn niet gelijk');
}
