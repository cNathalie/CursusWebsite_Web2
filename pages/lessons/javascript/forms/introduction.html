Forms: Introductie

Introductie Formulier Elementen Formulier Validatie Data Manipulatie Formulier Verzenden Oefeningen

In deze module gaan we het hebben over formulieren in JavaScript. Een formulier is een manier om gegevens van de gebruiker te verzamelen. Dit kan bijvoorbeeld een contactformulier zijn, een registratieformulier of een zoekformulier.

Formulieren bestaan uit verschillende elementen. Denk hierbij aan tekstvelden, selectievakjes, radio-buttons enzovoort. In deze module gaan we kijken naar de verschillende elementen die je kunt gebruiken, hoe je hun waardes uitleest, hoe je de input een formulier het beste valideert en hoe je een formulier verzendt.
