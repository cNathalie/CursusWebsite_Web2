JSON: Parse

Introductie JSON Syntax JSON Parse JSON Stringify Oefening

We kunnen JSON omzetten naar een JavaScript object.

JSON.parse()

JSON.parse() is een methode die een JSON-string omzet naar een JavaScript object.

Voorbeeld: 

          
// We hebben een JSON string
const json = '{"naam":"John", "leeftijd":30, "gehuwd":false}';

// We zetten de JSON string om naar een JavaScript object
const obj = JSON.parse(json);

// We kunnen nu de data manipuleren of gebruiken
console.log(obj.naam);
