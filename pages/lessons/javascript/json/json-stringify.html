JSON: Stringify

Introductie JSON Syntax JSON Parse JSON Stringify Oefening

We kunnen een JavaScript object omzetten naar een JSON-string.

JSON.stringify()

JSON.stringify() is een methode die een JavaScript object omzet naar een JSON-string.

Voorbeeld:


// We hebben een JavaScript object
const object = { naam: "John", leeftijd: 30, gehuwd: false };

// We zetten het JavaScript object om naar een JSON string
const json = JSON.stringify(object);

// De JSON string
console.log(json);
