Objecten: Date

De Date object is een ingebouwd object in JavaScript dat wordt gebruikt om datums en tijden te werken.

Date object aanmaken

Om een Date object aan te maken, gebruik je de new Date() constructor. De Date object constructor accepteert verschillende parameters, zoals jaar, maand, dag, uur, minuut, seconde en milliseconde.

Wanneer je geen parameters meegeeft, zal de Date constructor de huidige datum en tijd weergeven. 


// Maak een Date object aan
const date = new Date();
console.log(date);

// Maak een Date object met parameters aan
const myDate = new Date(1992, 09, 17, 09, 0, 0, 0);
console.log(myDate);

Wanneer je het Date object logt, zal het de datum en tijd weergegeven worden in een standaard format. Om dit om te zetten naar een leesbaar format, kun je gebruik maken van de verschillende Date object methodes.

.toLocaleString()

De .toLocaleString() methode wordt gebruikt om een datum/tijd te converteren naar een leesbaar format.

Wanneer je de methode zonder parameters gebruikt, zal de datum en tijd weergegeven worden in de taal en tijdzone van de gebruiker. 


// Maak een Date object aan
const date = new Date();
console.log(date.toLocaleString());

De .toLocaleString() methode accepteert ook parameters zoals locale en options. 


// Maak een Date object aan
const date = new Date();
console.log(date.toLocaleString('nl-BE', { weekday: 'long' }));

Met de locale parameter kies je in de code in welk format de datum moet worden weergegeven.
