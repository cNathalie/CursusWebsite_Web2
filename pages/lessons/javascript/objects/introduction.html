Objecten: Introductie

Introductie Objecten Aanmaken Object Properties Object Methodes Date Object Oefeningen

Objecten zijn een van de belangrijkste concepten in JavaScript. Een object is een zelfstandige entiteit met eigenschappen en type. In JavaScript is bijna alles een object. Een object kan een functie bevatten, die dan een methode wordt genoemd. Een object kan ook eigenschappen bevatten, die dan properties worden genoemd.

Objecten = Variabelen

Objecten zijn variabelen. Ze kunnen een waarde bevatten, maar in tegenstelling tot gewone variabelen kunnen objecten meerdere waarden bevatten. Deze waarden worden eigenschappen genoemd.

Voorbeeld:


// Een variabele
const car = "Fiat";

// Een object
const car = {
    type:"Fiat", 
    model:"500", 
    color:"white"
};
