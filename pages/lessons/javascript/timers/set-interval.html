De tweede timer functie is setInterval.

.setInterval()

Met de setInterval functie kunnen we een bepaald stukje code herhaaldelijk uitvoeren met een bepaalde interval. De functie accepteert twee argumenten: een functie die uitgevoerd moet worden en een getal dat aangeeft hoe lang we moeten wachten voordat de functie wordt uitgevoerd. Dit getal wordt uitgedrukt in milliseconden.

In het onderstaande voorbeeld wordt de functie countdown elke seconde uitgevoerd. De functie verlaagt de waarde van de variabele time met 1 en stopt wanneer de waarde 0 is.

10


// Declareer de variabele die de tijd bijhoudt
let time = 10;

// Declareer de functie die de countdown uitvoert
const countdown = () => {
    // Verlaag de waarde van time met 1
    time--;
    // Update de tekst van het element met id "time"
    document.getElementById("time").innerText = time;
};

// Declareer de Interval functie
const intervalFn = () => {
    setInterval(countdown, 1000);
};

// Haal de knop op
const button = document.getElementById("start");

// Voeg een event listener toe aan de knop
button.addEventListener("click", () => {
    // Start de countdown
    intervalId = setInterval(countdown, 1000);
});
