Timers: Set Timeout

Introductie Set Timeout Set Interval Clear Timers Oefeningen

De eerste timer functie is de setTimeout.

.setTimeout()

Met de setTimeout functie kunnen we een bepaald stukje code uitvoeren na een bepaalde tijd. De functie accepteert twee argumenten: een functie die uitgevoerd moet worden en een getal dat aangeeft hoe lang we moeten wachten voordat de functie wordt uitgevoerd. Dit getal wordt uitgedrukt in milliseconden.

In het onderstaande voorbeeld wordt de functie openGoogle pas uitgevoerd 5 seconden nadat er op de knop is geklikt. 


// Declareer de functie die Google zal openen
const openGoogle = () => {
window.open("https://www.google.com", "_blank");
};

// Declareer de TimeOut functie
const timeOutFn = () => {
     setTimeout(openGoogle, 5000)
};

// Haal de knop op
const button = document.getElementById("google");

// Voeg een event listener toe aan de knop
button.addEventListener("click", timeOutFn);

// Klik op de knop en tel tot 5...

Het kan zijn dat je browser pop-ups blokkeert. Als dit het geval is, zal de functie niet werken. Je kunt dit oplossen door de pop-up blokkering uit te schakelen.
