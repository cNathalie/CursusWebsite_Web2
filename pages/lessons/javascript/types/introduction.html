Types in depth: Introductie

Introductie Number Math Object String

In deze lessenreeks gaan we dieper in op de verschillende types in JavaScript. We zullen kijken naar de belangrijkste types die JavaScript ondersteunt en hoe je ze kan gebruiken in je code.

We zullen beginnen met het type Number en kijken naar hoe je getallen kan gebruiken in JavaScript. Daarna zullen we kijken naar het type String en hoe je tekst kan gebruiken in JavaScript.

Het is belangrijk om de verschillende types in JavaScript te begrijpen omdat dit je kan helpen om betere code te schrijven en bugs te voorkomen.

Dynamisch

JavaScript is een dynamisch getypeerde programmeertaal, of in het Engels: Dynamically Typed.

Dit wil zeggen dat er geen type wordt meegegeven bij het declareren van een variabele, functie, paramater of functie return waarde. Pas tijdens de uitvoering van de code wordt er gebruik gemaakt van types.

Dit in tegenstelling tot statisch getypeerde programmeertalen zoals C#, waarbij je bij het declareren van een variabele het type moet meegeven.

Types

Je kan het type van een variable achterhalen door gebruik te maken van de typeof() operator.

Plaats de lijst met variabelen in de console en vraag nadien één per één hun type op. 


// Declareer de variabelen
const leeftijd = 25;
const interest = 0.12;
const naam = "Jan";
const isStudent = true;
const print = (msg) => console.log(msg);

// Vraag nu van elke variabele zijn type op
typeof(// naam van de variabele);

// Vraag ook eens het type op van:
typeof(window);
typeof(document);

// Wat zou hier van het type zijn?
const kleur = new String("rood");
typeof(kleur);

Opgelet: Zoals je merkt is er een verschil tussen de types van naam en kleur, hoewel ze beiden een string bevatten. Dit komt omdat naam een string is en kleur een object is. Dit is een belangrijk verschil dat je moet onthouden.

Meer specifiek is kleur een object van het type String = een String object. Elke variabele die een waarde krijgt aan de hand van het keyword new is een object.

De string naam noemen we een string literal. Dit is een string die rechtstreeks in de code wordt geschreven tussen aanhalingstekens.
