Types in depth: Math Object

Introductie Number Math Object String

Het Math object is een ingebouwd object in JavaScript dat je toelaat om wiskundige operaties uit te voeren. Het object bevat verschillende methodes en eigenschappen die je kan gebruiken om berekeningen uit te voeren.

Random getallen

Met het Math object kan je random getallen genereren. Dit kan je doen met de Math.random() methode. Deze methode geeft een willekeurig getal tussen 0 en 1 terug.

Door het resultaat van Math.random() verder te manipuleren, kan je random getallen genereren binnen een bepaald bereik.

Probeer het uit:


// Een random getal tussen 0 en 1
const random1 = Math.random();
console.log(random1);

// Een random getal tussen 0 en 10
const random2 = Math.random() * 10;
console.log(random2);

// Een random getal tussen 5 en 50
const random3 = 5 + Math.random() * 50;
console.log(random3);

Afronden naar beneden

Je kan getallen afronden naar beneden met de Math.floor() methode. Deze methode rondt een getal af naar het dichtsbijzijnde gehele getal dat kleiner is dan of gelijk is aan het gegeven getal.

Probeer het uit:


// Afronden naar beneden van een positief getal
const number = 12.34;
const rounded = Math.floor(number);
console.log(rounded);

// Afronden naar beneden van een negatief getal
const negativeNumber = -12.34;
const negativeRounded = Math.floor(negativeNumber);
console.log(negativeRounded);

Afronden naar boven

Je kan getallen afronden naar boven met de Math.ceil() methode. Deze methode rondt een getal af naar het dichtsbijzijnde gehele getal dat groter is dan of gelijk is aan het gegeven getal.

Probeer het uit:


// Afronden naar boven van een positief getal
const number = 12.34;
const rounded = Math.ceil(number);
console.log(rounded);

// Afronden naar boven van een negatief getal
const negativeNumber = -12.34;
const negativeRounded = Math.ceil(negativeNumber);
console.log(negativeRounded);

Afronden naar het dichtstbijzijnde gehele getal

Je kan getallen afronden naar het dichtstbijzijnde gehele getal met de Math.round() methode. Deze methode rondt een getal af naar het dichtstbijzijnde gehele getal.

Probeer het uit:


// Afronden naar het dichtstbijzijnde gehele getal van een positief getal
const number = 12.34;
const rounded = Math.round(number);
console.log(rounded);

const number2 = 12.7;
const rounded2 = Math.round(number2);
console.log(rounded2);

// Afronden naar het dichtstbijzijnde gehele getal van een negatief getal
const negativeNumber = -12.34;
const negativeRounded = Math.round(negativeNumber);
console.log(negativeRounded);

const negativeNumber2 = -12.7;
const negativeRounded2 = Math.round(negativeNumber2);
console.log(negativeRounded2);

De Math.round() methode rondt af naar het dichtstbijzijnde gehele getal. Als het getal 0.5 of hoger is, wordt er naar boven afgerond. Als het getal lager is dan 0.5, wordt er naar beneden afgerond.

Maximum en minimum

Je kan het maximum en minimum getal van een reeks getallen bepalen met de Math.max() en Math.min() methodes.

Probeer het uit:


// Maximum getal
const max = Math.max(1, 2, 3, 4, 5);
console.log(max);

// Minimum getal
const min = Math.min(1, 2, 3, 4, 5);
console.log(min);
