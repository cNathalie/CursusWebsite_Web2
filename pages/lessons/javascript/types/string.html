Types in depth: String

Introductie Number Math Object String

Het type String in JavaScript wordt gebruikt om tekst te representeren. Een string is een reeks van karakters, zoals letters, cijfers, spaties, leestekens, enzovoort.

String declareren

Er zijn drie manieren om een String aan te maken:

  • Met enkele quotes: 'Hello, world!'
  • Met dubbele quotes: "Hello, world!"
  • Met backticks: `Hello, world!`

Je kan een String ook declareren met de String constructor: 


// Met het new keyword
const str = new String('Hello, world!');
console.log(str);

// Zonder het new keyword
const str2 = String('Hello, world!');
console.log(str2);

De simpelste, en aanbevolen manier is het aanmaken van een string literal. 


// String literal
const str3 = 'Hello, world!';
console.log(str3);

Concateneren

Soms is het nodig om dynamisch strings op te bouwen. Dit kan je doen door de + operator te gebruiken. 


// Concateneren met de + operator
const firstName = 'John';
const lastName = 'Doe';
const fullName = firstName + ' ' + lastName;
console.log(fullName);

Je kan ook concateneren met de methode concat():


// Concateneren met de concat() methode
const firstName = 'John';
const lastName = 'Doe';
const fullName = firstName.concat(' ', lastName);
console.log(fullName);

Je voelt aan dat bovenstaande manieren omslachtig en gevoelig voor bugs kunnen zijn. Het is daarom gemakkelijker en aanbevolen om een template literal aan te maken met backticks (``) en variabelen in te voegen met ${myVar}. 


// Template literal met variabelen
const firstName = 'John';
const lastName = 'Doe';
const fullName = `${firstName} ${lastName}`;
console.log(fullName);

String methodes

Strings hebben verschillende methodes die je kan gebruiken om manipulaties uit te voeren. Hier zijn enkele voorbeelden:

  • length: geeft de lengte van de string terug
  • toUpperCase(): zet de string om naar hoofdletters
  • toLowerCase(): zet de string om naar kleine letters
  • trim(): verwijdert spaties aan het begin en einde van de string
  • split(): splitst de string op in een array van substrings, je geeft mee op welk karakter de splitsing moet gebeuren.
  • replace(): vervangt een deel van de string door een andere string
  • charAt(): geeft het karakter op een bepaalde index terug
  • indexOf(): geeft de index van een bepaalde substring
  • lastIndexOf(): geeft de laatste index van een bepaalde substring
  • substring(): geeft een deel van de string terug, je geeft de start en eind index mee
  • slice(): geeft een deel van de string terug, je geeft de start en eind index mee

Probeer enkele van deze methodes uit:


// Een string met veel 'trailing spaces'
const str = 'Hello, world!     ';

// De lengte = het aantal karakters
console.log(str.length);

// Omzetten naar hoofdletters
console.log(str.toUpperCase());

// Omzetten naar kleine letters
console.log(str.toLowerCase());

// Spaties verwijderen
console.log(str.trim());

// Opsplitsen in een array
console.log(str.split(',')); 

// Vervangen van een deel van de string
console.log(str.replace('world', 'John'));

// Karakter op een bepaalde index
console.log(str.charAt(0));

// Index van een bepaalde substring
console.log(str.indexOf('world'));

// Laatste index van een bepaalde substring
console.log(str.lastIndexOf('world'));

// Deel van de string
console.log(str.substring(0, 5));

// Deel van de string
console.log(str.slice(0, 5));

Speciale karakters

Strings kunnen speciale karakters bevatten die je niet zomaar kan typen. Om deze karakters te gebruiken, moet je een escape karakter gebruiken: de backslash (\).

Sommige karakters maken deel uit van de programmeertaal en worden niet dus niet standaard letterlijk weergegeven in strings. Als je ze dus echt wil zien, moet je ze escapen.

Hier zijn enkele voorbeelden van speciale karakters:

  • \n: nieuwe lijn
  • \t: tab
  • \': enkele quote
  • \": dubbele quote
  • \\: backslash

Probeer enkele van deze karakters uit:


// Nieuwe lijn
console.log('Hello\nworld!');

// Tab
console.log('Hello\tworld!');

// Enkele quote
console.log('Hello\'world!');

// Dubbele quote
console.log('Hello\"world!');

// Backslash
console.log('Hello\\world!');
